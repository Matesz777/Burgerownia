# Burgerownia

A simple and visually focused restaurant website built with **HTML, CSS, and JavaScript**.  
The project presents a fictional burger brand experience with a landing page, highlighted products, and a separate menu page.

## Overview

Burgerownia is a static frontend project created to showcase a burger restaurant website with a bold visual style and straightforward user interaction.  
It focuses on layout, branding, menu presentation, and basic interface behavior using only core web technologies.

The project includes:

- a homepage with featured burgers,
- a dedicated menu page,
- interactive UI behavior with JavaScript,
- responsive styling for smaller screens,
- a restaurant-style promotional layout.

## Features

- **Landing page** with restaurant branding and hero-style presentation
- **Featured burgers section** with highlighted monthly items
- **Separate menu page** for browsing food and drink offerings
- **Interactive navigation behavior**
- **Dropdown-style mobile/side menu interaction**
- **Scroll-based navbar background effect**
- **Hover and selection effects** for product cards
- **Responsive layout adjustments** for smaller screen sizes

## Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
